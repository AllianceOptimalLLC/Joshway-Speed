# Joshway Speed

**Gorgeous Sonic-like 2D side-scroller with Joshway theme.**

Run, jump, collect star rings, explore beautiful parallax worlds, power-up and face challenges/boss. High speed platforming with heart.

## Live Demo
- **Production**: https://joshway-speed.vercel.app
- Vercel project: joshway-speed (AllianceOptimalLLC)

## Features
- 4+ stunning hand-crafted worlds (playground realms, backyard, moon, starfields)
- Smooth momentum physics, loops, speed boosts
- Collect rings for score + bonuses
- Power-ups: speed, cape flight/glide, magnet
- Epic boss encounter
- Chiptune music per level + SFX
- High scores, time bonuses, full production UI
- Keyboard + touch support

## Controls
- Arrow keys / WASD — Run left/right
- Space / Up — Jump (hold for higher)
- Down — Spin attack / roll
- Shift — Boost (when powered)

## Run Locally
```bash
npm install
npm run dev
npm run build
```
